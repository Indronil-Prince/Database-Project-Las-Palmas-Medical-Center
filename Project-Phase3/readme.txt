How to Run the Interactive Database Management Application
---------------------------------------------------------------------
This application allows users to manage a database through a web interface.


Prerequisites
---------------------------------------------------------------------
- Python 3.x installed on your system
- Flask framework installed (`pip install Flask`)
- MySQL server installed and running


Instructions
---------------------------------------------------------------------
1. Clone or download the repository to your local machine.

2. Navigate to the project directory using the command line.

3. Install the required libraries (Flask and mysql)

4. Set up the MySQL database and configure the connection details in `app.py`.

5. Run the Flask application:
    ```
    flask --app app run
    ```

6. Open a web browser and navigate to `http://localhost:5000` or `http://127.0.0.1:5000` to access the application.

7. Follow the on-screen instructions to use the application, including options to retrieve data, calculate averages, insert new records, delete records, and more.
